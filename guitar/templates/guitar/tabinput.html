How to use:

Firstly select which tuning you want to use from the Drop-down list. (At the moment only Standard Tuning is supported.)

Copy and Paste your favourite guitar tabs from sites such as Ultimate Guitar into the input box. It should look like:

Make sure the tab has your selected tuning in front of each tab line so the app can detect which string a line belongs to.

e.g. if you are in standard tuning, each line should begin with either "e|-", "B|-" etc...
